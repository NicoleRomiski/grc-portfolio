# Access Review Evidence Sample

## Objective
To demonstrate quarterly user access review process.

## Scope
Review of user accounts for:
- Google Workspace
- AWS IAM
- HR System

## Process
1. Exported user access list from each system.
2. Compared active accounts to HR roster.
3. Identified terminated employees with active access.
4. Submitted removal requests.

## Findings
- 2 terminated users retained AWS access.
- 1 contractor account exceeded contract end date.

## Remediation
- Access removed within 24 hours.
- Documented in ticketing system.

## Evidence Collected
- Access export CSV
- HR employee roster
- Screenshot of access removal confirmation
